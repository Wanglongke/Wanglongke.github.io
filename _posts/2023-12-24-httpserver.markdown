本地运行http服务

## Step1

安装 Node.js

## Step2

安装http-server
```
npm install -g http-server
```

## Step3

以管理员身份运行 powershell   
然后执行
```
set-ExecutionPolicy RemoteSigned
```
## Step4

```
http-server -p 3000
```
